## A. ☀ 태양광 시뮬레이션 (룩 확인용)

### A-1. 태양 + 시간 슬라이더
- `DirectionalLight` = 태양. 부드러운 그림자(shadow) 켜기.
- 슬라이더로 **시간대(아침 → 정오 → 저녁)** 조절:
  - 태양 **고도·방위** 변화 (아침 낮음 → 정오 높음 → 저녁 낮음, 방위도 이동).
  - **색온도** 변화: 아침 따뜻한 주황 → 정오 흰빛(중성) → 저녁 주황.
  - 하늘/환경광(ambient)도 시간 따라 살짝 연동(저녁이 좀 어둑).
- 태양이 창을 통과해 **바닥에 창틀 격자 그림자**가 떨어지고, 시간 따라 **그림자가 길어지고 이동**.

### A-2. 유리 종류별 빛 반응 (핵심 — 유리마다 다르게)
태양광이 유리를 통과할 때 바닥에 생기는 빛 패치가 유리종류마다 달라야 함:
| 유리 | 통과광(바닥) | 유리 표면 룩 |
|---|---|---|
| clear | 선명한 창격자 빛 패치, 밝음 | 맑고 살짝 반사·하이라이트 |
| frosted | **산란**돼 부드럽고 흐릿한 사각(격자 안 보임), 은은 | 뿌옇게 확산, 반사 약함 |
| green-tint | 통과광에 **살짝 초록빛** 틴트 | 옅은 초록 유리 |
| glassblock | 빛이 **조각조각 산란**, 격자블럭 패턴 | 두꺼운 산란 유리 |
| none | 빛 100% 통과(막힘 없음), 제일 밝음 | 유리 없음 |
- 유리 표면 자체의 **반사/하이라이트**도 태양각 따라 반짝이게(스펙큘러 or 가벼운 envMap).
- ★ 여기서 쓰는 확산/틴트는 **시각 효과(룩)**. transmittance 숫자(조도)는 이미 userData에 있고 로직 창이 나중에 씀 — 여기선 연결 안 함.

### A-3. 실내로 들어오는 느낌
- 창 안쪽(방 방향)에 **빛기둥/바닥 광 패치**. 큰 창(sunroom/apt_big)일수록 넓고 밝게, 작은 창(반지하 letterbox)일수록 좁고 약하게.
- 이걸로 "이 창이 방을 얼마나 밝히나"가 눈으로 느껴지게.

## B. 🎛 왼쪽 필터 패널 (붐빔 방지 핵심)

화면 왼쪽에 고정 컨트롤 패널. 위에서부터:

### B-1. ☀ 태양 시간 슬라이더 (맨 위, 항상 공용)
- 어떤 필터를 골라도 태양은 전체에 적용.

### B-2. 색상 필터 (라디오 — 하나만)
- `전체` / 흰 / 회색 / 흑(블랙) / 차콜 / 민트 / 베이지 / 우드 / 찐우드 / (추가 파스텔들)
- 고르면 **그 색 창만** 벽에 배치.

### B-3. 창 종류 필터 (라디오 — 하나만)
- `전체` / studio_cross / studio_grid / apt_big / semi_letterbox / bath_block / sunroom / academy_slim …
- 고르면 **그 종류만** 배치.

### B-4. 유리 필터 (체크박스 — 복수)
- clear / frosted / green-tint / glassblock / none. 체크된 유리만 표시.

### B-5. ★ 붐빔 방지 규칙 (제일 중요)
- **색·창 둘 다 "전체"면 135개 다 깔지 마.** → **창 종류당 대표색 1개씩(약 15개)**만 배치.
  (기본 진입 화면이 안 터지게)
- 색을 보고 싶으면 **창 종류를 하나 고르게** → 그럼 그 창의 **전 색상**이 벽에 펼쳐짐.
- 반대로 **색을 하나 고르면** → 그 색의 **모든 창 종류**가 펼쳐짐.
- 즉 "전체×전체 = 대표만, 한 축을 고정하면 다른 축 전개".

### B-6. 필터 동작
- 필터 바꾸면 벽에 해당 창들만 **다시 배치**(간격 균등), 카메라는 그 세트 폭에 맞게 정렬.
- 각 창 아래 라벨 유지: id · pattern · 색 · 유리 · 확정/초안.
- 창 많으면 벽을 옆으로 길게 + 카메라 좌우 패닝(드래그).

## C. 산출·확인
1. `_window_showroom.html`에 태양 슬라이더 + 왼쪽 필터 패널 추가.
2. 실시간 WebGL 유지(antialias:true, devicePixelRatio 반영 — PNG로 굽지 말 것, 그래야 선명).

4. 확인 포인트 짚어줘:
   - 태양 슬라이더 돌리면 그림자·색온도 바뀌나
   - frosted가 바닥 빛을 흐릿하게 산란하나 (clear랑 구분되나)
   - 큰 창이 더 밝은 빛기둥 만드나
   - 전체×전체에서 15개만 뜨나(안 터지나), 창 하나 고르면 색들이 펼쳐지나
