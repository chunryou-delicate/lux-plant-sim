# 준비물 마스터 체크리스트 — DB·에셋·시스템 (2026-07-12)

> 본격 구현 전, 준비할 것(에셋)·쌓을 것(DB)·만들 것(시스템)을 한눈에.
> ⭐ 원칙: **다 모으고 시작 ❌ → 스키마(구조) 먼저 + 대표 씨앗 → 구현하며 채운다.**
> (엔지니어 함정 경계: 데이터는 깊게, 기본플레이는 단순. 새 아이템 = 데이터 한 줄.)
>
> 표기: ✅완료 🟡부분 ❌없음 / 대상 [도구][게임][공용]

---

## 1. 데이터베이스 (dB)

### 1-1. 식물계
- ❌ [게임] **식물 종 DB** — 빛(PPFD/DLI·직사간접·무늬유지·광포화), 물(주기·과습), 온·습도·통풍, 성장(속도·단계·무늬발현·번식), 가치(희귀도·가격·수확), 병해충
- 🟡 [게임] 식물 **카테고리 정의** (꽃/관엽/허브/다육/난/분재 + 각 플레이축)
- ❌ [게임] 성장 **단계·곡선** 데이터
- ❌ [게임] **무늬 유전·확률표**
- ❌ [공용] **화분·용기 DB** (크기·배수·재질)
- ❌ [게임] 흙·배지 DB (심화)

### 1-2. 빛/조명
- 🟡 [공용] **조명기구 DB** — 종류·광속(lm)·배광·색온도(K) O / **스펙트럼(R:B·원적색)·lux→PPFD계수·소비전력·제품프리셋 ❌**
- ❌ [공용] **광원 스펙트럼 → PPFD 변환 테이블**
- 🟡 [공용] **자연광/창 DB** (방위·크기·유리투과율·차양) — 천공조도 단일값만
- ❌ [게임] **지역별 기상 DB** (일사·일조·운량·위경도·일출몰·계절)
- 🟡 [도구] **KS/권장 조도 기준** (용도별) — target 입력값만 있음

### 1-3. 공간/사물
- 🟡 [공용] **가구 DB** — 크기·높이·회전 O / **재질 반사율·가격·카테고리 ❌**
- 🟡 [도구] **표면 반사율 ρ DB** (나무·천·금속·유리…) — 슬라이더 값만, 재질 프리셋 ❌
- 🟡 [공용] **방/집 프리셋 DB** (원룸/투룸/L자·창위치·천장고) — 1개만
- 🟡 [공용] **방 구조 부품** (문·통로·창틀·선반·랙) — 데이터 구조만 열림

### 1-4. 게임 시스템 데이터
- ❌ [게임] 아이템 카탈로그(상점·가격·해금)
- ❌ [공용] **적합성 판정 임계값** (자리 PPFD vs 식물범위 → OK/부족/과다)
- ❌ [게임] 시간모드 배속표(이지/하드/영원)
- ❌ [게임] 계절·날씨 이벤트 테이블
- ❌ [게임] 도감(수집) 메타 / 업적·퀘스트 / 경제·밸런스 테이블

### 1-5. 기타
- ❌ [공용] 다국어(로컬라이제이션) 문자열
- ❌ [공용] 세이브 데이터 스키마
