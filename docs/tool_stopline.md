# 도구 종료선 & 경쟁 지형 (2026-07-12 확정)

> 오늘의 핵심 결정: 조도 도구를 "어디까지만" 하고 게임으로 넘어갈지 선 긋기.
> 이유: 조도 도구 시장은 레드오션. 여기서 정밀도·기능 경쟁하면 그 밑에 한 명 더 될 뿐.
> → 다음에 멋진 경쟁 도구를 봐도 이 문서로 돌아와 안 흔들릴 것. ⭐

---

## 1. 왜 이 선을 긋나 (냉정한 팩트)

```
확인된 사실:
- 웹 조도 계산기 = 이미 많음 (Luxiflux, Visual, lightingCalc, EleCalc…)
- 제조사마다·용도마다 있음 (Acuity Visual: 실내/도로/벽/주차장/투광…)
- 대부분 lumen method(평균값·사각형·빈방) → 우리가 이미 더 정밀
- DIALux/Relux = 무료·최강·정밀 → 정면승부 불가
- 근데 이것들 전부 "전문가가 조명 선택·견적 내는 도구"

우리가 조도 도구로 승부하면:
- 제조사 무료 도구 + DIALux 다 있는데서 "웹이라 쉬움" 하나로 버팀
- 아무리 정밀하게 만들어도 = 그 밑에 한 명 더

→ 결론: 도구 정밀도·기능은 DIALux 벽을 못 넘음. 인정.
        도구는 "게임의 엔진"까지만. 그 이상은 함정.
```

---

## 2. 경쟁 지형 지도 (3층)

```
        정밀 ↑
             │  DIALux · Relux · Visual Pro · AGi32
             │  (무료~유료, 정밀, 설치, 전문가, IES/CAD)
             │  ← 절대 안 됨. 정면승부 없이.
   ──────────┼──────────────────────────────
             │  ★ 우리 자리 ★
             │  웹 + 자리별조도(point) + 자연 채광 + 가구
             │  + "식물 연결" (아무도 없는 축)
   ──────────┼──────────────────────────────
             │  웹 lumen 계산기들
             │  (Luxiflux, Visual Interior, lightingCalc…)
             │  평균값·사각형·빈방 → 우리가 이미 넘어섬
        간단 ↓  (제조사 선택 견적용, 일반인용 아님)

   ※ 식물 축은 아무도 없음:
     - 조명업계: 제품 선택·판매가 목적, 식물 관심 0
     - 식물앱: 조도 물리계산 0 ("밝은곳/그늘" 수준)
     - 교집합 = 빈 곳 = 우리 자리
```

## 3. 도구 종료선 (여기까지만 = 게임 엔진 완성)

```
✅ 할 것 (전부 게임에 이어짐 = 헛일 아님)
├─ 방 그리기 (벽 미리보기)      → 게임 방 만들기
├─ 가구 드래그·회전             → 게임 방 꾸미기
├─ 조명 배치                   → 게임 조명·식물등
├─ 조도 히트맵·자리별 밝기      → 이미 lumen 계산기보다 나음
└─ 계산면 높이 (선반별 조도)    → 식물 "이 높이 되나"에 직결

━━━━━━━━━━ 여기서 멈춤 ━━━━━━━━━━

❌ 안 할 것 (정밀·기능 = 함정 = 봐도 안 흔들림)
├─ 용도별 계산기 쪼개기 (도로·벽·주차장… Visual식)
├─ 제품 IES 데이터 연동
├─ 등조도 곡선 (isolines)
├─ 리포트 출력 (PDF)
├─ 자동 조명 개수 역산
├─ 에너지 지표 (W/㎡)
├─ CAD/BIM 임포트
└─ DIALux와 정밀도 경쟁
   ※ 이것들 = 전문가용. 우리 타겟(식집사·식물애호가) 아님.
   ※ 정 필요하면 먼 훗날 트랙A에서. 지금은 절대 아님.
```

## 4. 흔들림 방지 체크 (경쟁 도구 볼 때마다)

멋진 조도 도구를 보면 이 질문만:
```
Q1. 이게 "빛→식물"을 하나?      → 거의 100% 아니오 → 우리 축 아님, 넘어가
Q2. 이게 lumen method인가?      → 대부분 예 → 우리가 이미 넘어섬
Q3. 이게 전문가 견적 도구인가?  → 대부분 예 → 우리 타겟 아님
Q4. 그럼 나도 저거 해야 하나?   → ❌ 종료선 밖이면 안 함
```

**핵심 주문:** "조도 도구는 이미 세상에 충분하다. 우리 무기는 도구가 아니라 식물이다."

## 5. 그래서 다음 순서 (확정)

```
① 방 그리기 마무리    → 클코 진행 중
② 가구 드래그        → 다음
③ (조명 약간 다듬기)
④ 계산면 높이
──── 도구 종료 ────
⑤ ★ 식물 여기 ★ → "빛이 진짜 → 식물 반응" 첫 다리
   = 게임 시작 = 진짜 무기
```
