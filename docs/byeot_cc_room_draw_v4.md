# 클코 작업 요청 — 방 그리기 사용성 최적화 (v4)

> tool.html 의 방 그리기를 "막막하지 않게" + 나중 확장(문·통로·다중방) 대비 구조로.
> 타겟: 적당히 아는 식집사·애호가 (바보 아님, 근데 CAD도 아님).
> 정리: 2026-07-12

---

## 0. 왜 지금 이걸 하나

- 도구의 첫 관문 = 방 입력. 여기서 막히면 아무것도 못 함.
- 지금 방 그리기(캔버스 클릭)는 기능은 되나 **"뭘 만드는지 안 보여서 막막"**.
- 핵심 해결: **그으면서 벽이 실시간으로 서는 게 보이게.**
- 동시에: 나중에 문·통로·다중방 넣을 때 갈아엎지 않게 **데이터 구조를 미리 열어둠.**

---

## 1. 현재 상태 (tool.html 에 이미 있음)

```
✅ 자유형 폴리곤 그리기 (S.draw.pts 에 꼭짓점 배열)
✅ Shift=직각, 0.1m 격자 스냅, 꼭짓점 흡착(포인트 스냅)
✅ 뒤로가기 (Backspace / 우클릭 → pts.pop())
✅ updateDrawPreview() → 바닥에 노란 선 + 꼭짓점 점만 그림
✅ 시작점 근처/더블클릭 = 완료 → S.poly 확정
✅ 창(window) 기능도 이미 있음 (벽 클릭해서 배치)
```

→ 뼈대는 좋음. 문제는 "선만 보이고 벽이 안 보임" + "구조가 가구부·다중방 대비 안 됨".

---

## 2. 이번에 할 것 = ① 방 그리기 (실행) + 구조 열어두기

### 2-1. 벽 실시간 미리보기 (막막함 해결) ← 핵심
```
그리는 중:
- 찍은 점들 사이 = 확정된 벽, 반투명으로 실시간 서기 (한붓그리기로 쌓임)
- 지금 마우스 따라가는 변 = 미확정 벽 (더 옅은 반투명)
- 벽 높이 = S.H(천장고)만큼 올라감 → 방 크기 체감
- 뒤로가기(Backspace/우클릭) = 직전 점 + 그 벽 같이 사라짐
완성(닫으면):
- 4면(이상) 벽 확정 → S.poly 저장 → 기존 buildRoom() 로직
```
- 구현: updateDrawPreview() 를 선+점만 → **반투명 벽 쿼드 추가**로 확장.
- ⚠ 시야 가림 방지: opacity 낮게(0.2~0.3), 미확정 벽은 더 옅게.

### 2-2. 데이터 구조 = 가구부·다중방 대비 (지금은 안 씀) ← 중요
```
지금 구조:  S.poly = [{x,z}, ...]           // 방 하나, 외곽선만
바꿀 구조:  S.rooms = [ {                    // 방 여러 개 대비 (지금은 1개만 씀)
              poly: [{x,z},...],             // 외곽선
              walls: [ {                      // 벽마다 = 한 변
                a:{x,z}, b:{x,z},             // 시작·끝
                openings: [                   // ← 가구부 리스트 (지금은 빈 배열)
                  // {type:'door'|'passage'|'window', t, width, sill, head}
                ]
              }, ... ]
            } ]
```
- **지금은 openings 빈 배열로 두고 방 하나만.** 기능은 ❌ 만.
- 근데 구조가 이미 문·통로·창을 품을 수 있음 → 나중 ②③ 칠 때 덜 갈아엎음.
- 기존 S.windows 도 이 openings 로 흡수(마이그레이션 주석).
- ⚠ 기존 조도 계산(pointIllum, daylightAt)이 S.poly 참조 → S.rooms[0].poly 로 연결만.

### 2-3. 프리셋 방 (막막함 2차 해결, 선택적)
```
빈 화면 "그리세요"가 막막 → 프리셋 버튼 먼저:
- [네모] [원룸] [L자] → 기본 방 나옴 → 크기만 조정
- 그리기는 "직접 그리기(고급)"로
※ 여유 되면. 우선순위 2-1 > 2-2 > 2-3.
```

---

## 3. 단계적 로드맵 (이번엔 ① 만, 구조는 이어둠)

```
【① 방 그리기】 ← 이번
  - 벽 실시간 미리보기 (한붓그리기, 반투명 벽, 뒤로가기)
  - 데이터 구조를 openings·다중방 대비로
  - (여유) 프리셋 방

【② 벽 뚫기 (가구부)】 ← 다음
  - 완성된 벽 클릭 → 문/통로/창 선택해서 뚫기
  - 빛이 가구부로 통과 (조도 계산 반영)
  - 구조(2-2)가 이미 받으므로 UI만 추가

【③ 다중 방·통로】 ← 그 다음
  - 방 하나 더 그리기 → L자·복도 (S.rooms 에 폴리곤 추가)
  - 방 사이 통로로 빛 연결
```

---

## 4. 주의 (흔들리지 말 것)

- **기본은 단순 유지.** 화면은 "방 그리세요" 하나. 구조만 깊게.
- 기존 조도 엔진 안 건드림. 방 데이터 참조 경로만 갈아끼워서.
- 벽 뚫기·다중방은 이번에 UI 구현 ❌ (구조만). 함정 경계.
- 각 단계 브라우저 확인 후 다음.
