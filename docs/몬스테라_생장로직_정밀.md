# 볕 — 몬스테라 성장 로직 정밀 정의 (2026-07-16)

> 박사님이 다이어그램으로 정의한 실제 성장 규칙. 클코 구현용 스펙.
> ⚠ 이전 오해 교정: petiole는 잎자루 아님 / 줄기는 계속 안 굵어짐(고정)

---

## ■ 부품 정의 (에셋 매핑)

### 줄기 (굵기 = 종류별 고정 범위, 계속 안 자람)
```
petiole      = 빨강. 이른의 어린 줄기 (←잎자루 아님!)
               유일하게 길이 변함 (아래 "petiole 거동" 참조)
stem_thin    = 분홍 마디 (가는 쪽)
stem_med     = 분홍 마디 (중간)
stem_thick   = 초록 마디 (굵음, 아구리 혹 있음)
stem_main    = 하늘 마디 (희귀 확률)
```

### 잎/새순 (성장 단계 색상)
```
주황  = bud_furled     (말린 새순, 처음)
노랑  = bud_opening2   (펴지는 중)
초록  = leaf_young     (어린 통잎)
파랑  = leaf_mid1/mid2 (중간잎, 생성 시 랜덤으로 1 or 2)
보라  = leaf_mature    (완전 성숙잎, 최종)
```
성장 순서:
```
bud_furled → bud_opening2 → leaf_young → leaf_mid(1|2 랜덤) → leaf_mature
```

---

## ■ 마디 상태 전이 (성장의 핵심)

```
[분홍 마디] = 대기 상태 (stem_med 또는 stem_thin, 랜덤)
     │       평범한 줄기 마디. 아직 새 가지 안 남.
     │
     │  ← 분홍들 중 랜덤으로 1개 선택되어 전이
     ▼
  ├─ 보통 → [초록 마디] (stem_thick)
  │         아구리 혹에서 petiole(빨강, 새 줄기) 돋음
  │
  └─ 드물게 → [하늘 마디] (stem_main)
            희귀 확률 뽑기 + (초록처럼 혹에서 새 줄기)
```

핵심:
- 마디는 처음부터 종류 고정 X. **분홍(대기) → 초록/하늘(활성)로 전이**
- "아무 마디나 랜덤 발생" = 분홍 중 랜덤 선택
- 굵은 마디(초록=stem_thick)의 혹에서 새 줄기가 돋음 = 실제 몬스테라와 동일

---

## ■ petiole (빨강) 거동 — 유일하게 변하는 줄기

```
1. 초록/하늘 마디의 혹에서 petiole(빨강) 돋음
2. petiole가 자라며 길어짐 (유일하게 길이 변하는 줄기)
3. 일정 길이 도달 → petiole 2개로 분화
   (아래 만큼이 새 줄기 2마디로 굳어짐)
4. 즉 [빨강 늘어남 → 빨강 2개로 나뉨]
5. 굳어진 petiole → 분화 마디(stem_med/thin, 랜덤)가 됨
   → 다시 대기 마디로, 여기서 또 새 가지 될 수 있음 (반복)
```

성장 순환:
```
분홍(대기) → 초록(활성, 혹) → petiole(빨강, 새줄) → 길어짐 → 2개로 분화
  → 굳어져 분화(대기) → ...
동시에 petiole 돋은 곳에서 잎 발생 → 성숙 단계 진행
```

---

## ■ 잎 성숙과 줄기 (별개로 병렬 진행)

```
petiole 돋은 곳에 새순(주황) 발생
  → bud_opening2(노랑) → leaf_young(초록) → leaf_mid1/2(파랑, 랜덤) → leaf_mature(보라)
```
- 잎은 시기 단계대로 성숙 / 줄기는 마디 상태 전이대로 / 둘은 병렬

---

## ■ 랜덤 4요소 (개체마다 다른 이유)

```
1. 어느 분홍 마디가 활성화되는가 = 랜덤 → 가지 위치 제각각
2. 초록(보통) vs 하늘(드묾) = 랜덤
3. 분홍이 stem_med냐 stem_thin이냐 = 랜덤
4. 파란 잎이 mid1이냐 mid2냐 = 생성 시 랜덤
→ 4가지 랜덤이 겹쳐 "세상에 하나뿐인 개체" (시드로 고정)
```

---

## ■ 이전 시스템과 차이 (수정 포인트)

```
✗ 기존: 시간 지나면 계속 굵어지고 길어짐
⭐ 수정: 줄기는 종류별 고정 범위 / petiole만 길이 변함(길어졌다 2개 분화)
✗ 기존 오해: petiole = 잎자루  →  ⭐ petiole = 이른 어린 줄기
✗ 기존 오해: 마디 종류 처음부터 고정  →  ⭐ 분홍(대기)→초록/하늘(활성) 전이
```

---

## ■ 희귀 (하늘 마디 stem_main) — 나중 Phase (지금 자리만)
```
하늘 마디 = 희귀 확률 뽑기 지점 → 경제 2층(샵링크)에서 무늬종 연결 예정.
지금은 자리만, 실제 희귀종 로직은 나중.
```

## ■ 클코 구현 주의
```
- 줄기 굵기: 종류별 고정(scale 좁게), 계속 안 키우기
- petiole: 길이 애니 → 자라고 길어 → 2마디 분화 → 분화 전이
- 마디 상태: 분홍 배열 관리 → 랜덤 pick → 초록/하늘 전이 → 혹에서 petiole
- 잎 성숙: 별도 타이머로 단계 전이(줄기와 병렬)
- 랜덤 4요소: 시드 기반(개체 고정)
- 혹 위치: stem_thick 아구리(가시 혹)
```
