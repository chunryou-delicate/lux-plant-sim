## 🔆 진행 중 — 자연광(창·채광)
- [ ] 창 배치: 방 벽(폴리곤 변)에 클릭으로 창 추가
- [ ] 창 목록: 폭·높이(sill/head)·위치 편집, 삭제
- [ ] 채광 물리: 천공 조도 → 실내 조도 공간분포 (창가 밝고 안쪽 어두움)
      · 창을 면광원으로 보고 작업면 각 점에 적분
- [ ] 방위(N/E/S/W)·시간대·맑음/흐림 → 천공 조도 산출 (단계적)
- [ ] 인공광 + 자연광 합산 → 평균/균제도/판정에 반영
- [ ] 3D: 벽에 창 개구부 표시
- [ ] (심화) 창 통한 직사일광 = 바닥 밝은 패치 + 그림자

## 💡 조명 심화
