## 🔴 종료선 밖 = 안 함 (경쟁 도구 봐도 흔들리지 말 것)

> tool_stopline.md 참고. 이건 전문가용 = 우리 타겟(식집사) 아님.
> DIALux·Visual 등이 이미 다 함. 정면승부는 (정 필요하면) 먼 훗날 트랙A로.

```
✂ CAD/BIM 임포트 (DXF/DWG/IFC)  → 일반 타겟은 도면 없음
✂ 리포트 출력 (PDF)             → 시공사 산출물용
✂ 등조도 곡선 (isolines)        → 전문가 신뢰도용
✂ 자동 조명 개수 역산           → 실무 견적용
✂ 에너지 지표 (W/㎡)            → 실무용
✂ 실제 제품 IES DB·프리셋       → 제조사 견적 도구. 우리는 배광 프리셋 근사
✂ 용도별 계산기 쪼개기          → 도로·벽·주차장 (Visual식). 범위 아님
✂ DIALux 정밀도 경쟁            → 못 이김. 인정.
```

**흔들림 방지 주문:** "조도 도구는 이미 세상에 충분하다. 우리 무기는 도구가 아니라 식물이다."
